# WinFormsFigma.Tool

Starter repository for a reusable .NET Framework 4.7.8 class library.

## Prerequisites

- .NET SDK 9.0+

## Build

```bash
dotnet build src/WinFormsFigma.Tool/WinFormsFigma.Tool.csproj
```

## Pack

```bash
dotnet pack src/WinFormsFigma.Tool/WinFormsFigma.Tool.csproj -c Release
```
